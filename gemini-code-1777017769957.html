⚠️ 需要验证的 2 个重点

🎯 重点 1：左侧 W2 修补痕（2 处）

是什么： W2 = 板金修复后有中等程度的波浪纹或补漆痕。不是事故车等级（事故车会标 R / RA / XX），但确认有做过修复。

为什么要查： 4.5 分车身 + 左侧两处 W2 通常代表小范围钣金美容，但要排除"侧面碰撞后局部修复"的可能性。

查这 3 层：

  • 表面层（阳光下看）: 左侧车身 45° 斜角，反光有没有波浪纹；左前门、左后门颜色和其他门有没有色差；门缝上下宽度是否均匀。
  • 结构层（最关键 — 决定买不买）: 门 hinge 螺丝（有没有拧动过的痕迹）；B 柱、车门内侧（有没有焊接/修复痕迹）；左侧 side sill（门槛下）+ 底盘（有没有切割焊补）；发动机舱左侧纵梁（有没有敲打/焊接）。
  • 功能层（实车测试）: 左侧双滑门开关 10 次以上，听声音；左前/左后门关门声音与右侧对比；车门开合阻尼感觉。
判断标准：
三层都干净 → W2 只是表面补漆，可以买
结构层有动过 → 不买，可能是侧撞修复车
